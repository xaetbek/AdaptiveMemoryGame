Introduction:

Welcome to the adaptive memory game. In this game, you will be presented with a set of words which you have to memorize in the given 1 seconds. For every word you entered correctly, you will earn 1 point. Otherwise, vice versa. Also, you will be deducted a point if you do not match the number of challenged words. Finally, you will be a winner once you reach 20 points.

Instructions:

The game will prompt you to start the game where you have to enter Y. Then, You will be presented with a challenging set of words. After a second when the words disappear, you have to enter the words you have memorized. After, you will be shown you current score and the prompt ready. Once you reach 20 points, you will be shown final score.
